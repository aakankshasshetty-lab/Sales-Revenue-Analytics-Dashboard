# Sales-Revenue-Analytics-Dashboard
 `End-to-end sales and revenue analysis using Excel, SQL, and Power BI

📌 Project Overview
This project presents an end-to-end **Sales & Revenue Analytics Dashboard** built to analyze business performance using structured sales data. The objective is to identify key trends, high-performing products, and regional profitability to support data-driven decision-making.

The project demonstrates a complete data analytics workflow — from raw data cleaning to SQL-based analysis and interactive dashboard creation.

🛠 Tools & Technologies
- **Excel** – Data cleaning, preprocessing, and duplicate removal  
- **PostgreSQL** – Data storage, validation, and analytical SQL queries  
- **Power BI** – Interactive dashboard creation and data visualization  

📂 Dataset Description
The dataset contains transactional sales data with the following attributes:
- Order and shipping details
- Product category and sub-category
- Sales, profit, quantity, and discount
- Customer segment and regional information

🔍 Key Analysis Performed
- Calculated core KPIs: **Total Sales, Total Profit, Quantity Sold, Profit Margin**
- Analyzed sales performance across product categories
- Evaluated regional profitability
- Identified **Top 10 sub-categories** by sales contribution
- Assessed the relationship between discounts and profitability

📊 Dashboard Overview
The Power BI dashboard provides:
- KPI cards for quick business insights
- Comparative bar charts for category and region analysis
- Product-level performance visualization
- Profitability assessment using scatter analysis
- Interactive filtering using slicers


🚀 Business Insights
- Certain product categories contribute disproportionately to total revenue
- Regional performance varies significantly in terms of profitability
- Higher discounts generally correlate with reduced profit margins
- A small number of sub-categories drive a large share of total sales
